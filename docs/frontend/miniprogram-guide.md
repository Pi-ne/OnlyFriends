# OnlyFriends 小程序前端使用与测试教程

本文用于说明如何启动后端、打开微信小程序前端，并通过前端界面完成主要业务测试。

## 1. 前端位置

微信小程序源码目录：

```text
frontend/miniprogram
```

请在微信开发者工具中导入该目录，不要导入 `frontend` 根目录。

## 2. 启动后端

在项目根目录执行：

```powershell
.\scripts\start-all.ps1 -WithAi -Background
```

后端统一网关地址：

```text
http://localhost:8080
```

小程序 API Base URL 已配置为：

```text
http://localhost:8080/api/v1
```

如果微信开发者工具中请求本地接口失败，请在开发者工具中开启：

```text
不校验合法域名、web-view、TLS 版本以及 HTTPS 证书
```

## 3. 打开小程序

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择：

```text
frontend/miniprogram
```

4. AppID 可使用测试号或开发者工具测试 AppID。
5. 编译运行。

## 4. 推荐测试流程

### 4.1 登录注册

1. 进入“我的”。
2. 点击“去登录”。
3. 没有账号时点击“去注册”。
4. 注册后需要激活账号，激活 token 可从数据库或后端日志中获取。
5. 激活后返回登录页，用邮箱和密码登录。

### 4.2 活动核心闭环

1. 进入“发现”。
2. 查看活动列表。
3. 使用搜索框或筛选条件查询活动。
4. 点击活动卡片进入详情页。
5. 点击“立即报名”。
6. 返回“我的” -> “我的活动”，查看报名记录。
7. 回到活动详情，点击“取消报名”验证取消报名。

### 4.3 发起活动

1. 进入底部 Tab“发起”。
2. 填写基础信息、时间地点、规则、说明。
3. 可点击“AI 帮我策划”自动补充标题、标签或说明。
4. 点击“保存草稿”验证草稿创建。
5. 点击“提交审核”验证活动提交审核。

说明：后端 `POST /activities` 在 `isDraft=false` 时会直接进入审核流程，前端不会额外调用 `/activities/{id}/submit`。

### 4.4 小队

1. 进入“小队”。
2. 查看“我的小队”和推荐小队。
3. 点击“创建”新建小队。
4. 进入小队详情后测试加入、退出、成员、动态、相册、文件、投票、积分等入口。

### 4.5 好友与社交

1. 进入“我的”或小队/社交入口。
2. 打开好友页面。
3. 查看好友、关注、粉丝和好友申请。
4. 对好友申请执行同意或拒绝。

### 4.6 消息

1. 进入“消息”。
2. 查看会话列表。
3. 点击会话进入聊天页。
4. 输入文本并发送。

## 5. 已验证的后端联通情况

已通过后端 smoke 测试覆盖：

- 网关公开活动列表
- 用户注册、激活、登录
- 创建活动
- 附近活动查询
- 活动报名
- 活动签到
- 活动评价
- 关注和好友申请
- 创建和加入小队
- 私聊、群聊、会话已读
- 管理员登录和基础管理接口

## 6. 常见问题

### 6.1 页面提示网络异常

请确认网关可访问：

```powershell
Invoke-WebRequest "http://localhost:8080/api/v1/activities?page=1&size=1" -UseBasicParsing
```

### 6.2 需要登录的页面跳到登录页

这是预期行为。登录成功后会回到原页面或回到首页。

### 6.3 注册后无法登录

后端注册默认需要邮箱激活。开发阶段可从数据库中查询 `activate_token`，访问：

```text
http://localhost:8080/api/v1/auth/activate?token=你的激活token
```

### 6.4 地图或附近活动失败

请确认微信开发者工具已允许定位权限，并且后端服务正常运行。

## 7. 关闭服务

后台启动后可执行：

```powershell
.\scripts\stop-all.ps1
```
