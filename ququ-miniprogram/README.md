# Onlyfriends 微信小程序前端原型
i
这是一个原生微信小程序原型工程，可直接用微信开发者工具导入。

## 导入方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择 `codes/ququ-miniprogram`。
4. AppID 可先使用测试号或游客模式，当前 `project.config.json` 使用 `touristappid`。

## 已覆盖页面

- 首页：推荐、最新、附近活动 Tab，搜索与地图入口。
- 搜索页：关键词搜索、活动类型筛选、空状态。
- 活动详情页：活动信息、发起人、安全须知、报名确认。
- 发布页：活动模板、AI 策划草稿、保存草稿、提交审核。
- 地图模式：活动点位摘要与详情跳转。
- 社群页：兴趣小队列表与加入申请。
- 消息页：活动提醒和会话入口。
- 我的页：资料、报名/发布统计、商家申请入口。
- 登录/注册页：邮箱账号流程原型。

## 接口接入位置

- `app.js`：配置 `globalData.apiBase`，默认 `http://localhost:8080/api/v1`。
- `utils/request.js`：统一封装 `wx.request` 与 JWT 请求头。
- `api/activity.js`：活动、报名、AI 策划接口。
- `api/user.js`：登录、注册、个人资料接口。
- `data/mock.js`：当前页面使用的原型假数据。

后端连通后，优先替换首页列表、详情、登录、发布提交和报名接口。
